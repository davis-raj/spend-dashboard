# 💰 Family Spend Dashboard

A self-updating spend tracker. Drop a CSV, push, and the dashboard updates automatically.

## Live Dashboard

👉 **https://YOUR_USERNAME.github.io/spend-dashboard/**

Share this link with anyone — it always shows the latest data.

## How to Update (monthly, ~30 seconds)

1. Export your transactions CSV (from Copilot / your bank app)
2. Name it like: `Transactions - Jun 2026.csv`
3. Drop it into the `data/` folder in this repo
4. Commit & push (or upload via GitHub.com web UI)

The GitHub Action auto-rebuilds `data.json` and redeploys the site.

## Initial Setup (one-time)

1. Create a new GitHub repo called `spend-dashboard`
2. Push this folder to it:
   ```bash
   cd ~/spend-dashboard
   git init
   git add .
   git commit -m "Initial spend dashboard"
   git remote add origin https://github.com/YOUR_USERNAME/spend-dashboard.git
   git push -u origin main
   ```
3. In repo Settings → Pages → Source: select "GitHub Actions"
4. The first deploy runs automatically on push

## Local Preview

```bash
python3 build.py
cd docs && python3 -m http.server 8000
# Open http://localhost:8000
```

## File Structure

```
data/                    ← Drop monthly CSVs here
docs/index.html          ← The dashboard (auto-deployed)
docs/data.json           ← Generated by build.py (don't edit manually)
build.py                 ← Merges CSVs → data.json
.github/workflows/       ← Auto-build on push
```
